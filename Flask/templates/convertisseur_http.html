Convertisseur de fichier

Transforme un fichier en un fichier JSON au format {métadonnées, fichier_binaire}

Formats acceptés :

pdf, jpeg, png, jpg, gif, bmp, txt, py, csv, ods, odt, odg, ipynb, json, docx, doc, xls, tex

Si votre fichier est corrompu, vous n'obtiendrez pas les métadonnées particulières liés à l'extension du fichier considéré.
